Validation Of Events - Iv

Instructions

Overview

In this job, you are required to review a few webpages in order to answer questions based on their content.

For this job we define an EVENT in terms of the ENTITIES (either a physical, social, or mental object) involved in it, within a particular time interval.
For example, { Andy Murray;David Ferrer;2011 Australian Open; } in the time interval 18-01-2011 to 07-02-2011.

  • ENTITIES are separated using a semi-colon ';' as shown in the example { Andy Murray;David Ferrer;2011 Australian Open; }.
  • Based on your responses to the simple questions corresponding to each unit, we aim to validate a set of EVENTS thereafter.

Rules and Tips

Kindly adhere to the instructions provided in order to complete the task successfully.

  1. For each unit, you will be presented with a set of ENTITIES, a time interval, and a URL to a webpage.
  2. Follow the URL and observe the content in order to answer the corresponding questions.
  3. Please do not use your own knowledge about an event, but rely only on the evidence found on the Webpage.

*TRUE EVENT *

  1. In case one or more entities appear on the Webpage and correspond to an event within the time interval then it is a true event.
  2. The number of entities corresponding to the event strictly depends on those entities that are clearly within the time interval of the event.
  3. Please note that partial information about the time of an event is insufficient in establishing a TRUE EVENT. (For example, January 2011 is not sufficient and such instances can be marked with '-2' to signify temporal ambiguity)

NOT A TRUE EVENT

  1. In case one or more entities appear on the Webpage but do not correspond to the same event, and none of the events lie within the time interval.
  2. In case one or more entities appear on the Webpage, correspond to the same event, but do not correspond to the given time interval.

ENTITIES : { Andy Murray;David Ferrer;2011 Australian Open; }
Time Interval : 18-01-2011 to 07-02-2011
URL : http://www.theguardian.com/sport/2011/jan/26/australian-open-andy-murray-david-ferrer

Consider the following unit :

ENTITIES : { Natalie Portman; Academy Award for Best Picture; 83rd Academy Awards;}
Time Interval : 18-01-2011 to 07-02-2011
URL : Click Here

In the above example, note that from the content of the webpage we can see that not all the entities are evidently taking part in the event within the REQUIRED time interval: only "83rd Academy Awards" is explicitly related to a date (January 25, 2011) within the required time period. Therefore, the response to the question,'How many of the entities conform to the same event within the given range of 18-01-2011 to 07-02-2011? ', should be '1'.

For each unit :

  • Copy the corresponding URL into your browser and hit ENTER.
  • Use hot-keys to find whether the ENTITIES are present (CTRL+F can be used to search the webpage).
    In case of the example mentioned above, you should search for "Andy Murray", "David Ferrer", and "2011 Australian Open".
  • NOTE that you can use your discretion to determine whether or not partial mentions of entities such as "Murray" refer to the given entity, "Andy Murray". If there is enough clarity that the entity mentioned is present, then it does not mandatory that the entire entity name appears in the webpage. Other examples include, "Best Actress Academy Award" being equivalent to "Best Actress" if the page is talking about "Academy Awards".
  • Similarly, check if any plausible EVENT context is within the prescribed time interval. By exploring the content of the webpage, it is clear that Andy Murray and David Ferrer played in the 2011 Australian Open, and would face each other in the semi-final, therefore all the entities conform to the same event.

Be Careful Of

  • NOTE that the context of the entities that are found on a webpage is of prime importance. That is, read the surrounding sentence(s) that involve the entities. For example consider the following unit :

ENTITIES : { Rey Mysterio;Kevin Nash;Elimination Chamber; }
Time Interval : 18-01-2011 to 07-02-2011
URL : Click Here

In the above example, note that from the content of the webpage we can see that the entities do not participate in the same event. Therefore, the response to the question,'How many of the entities conform to the same event within the given range of 18-01-2011 to 07-02-2011? ', should be '0'.

  • Use CTRL+F to help as a tool in finding the entities and the surrounding context, but not as merely a tool to finish the job. NOTE that you can use your discretion in order to identify an entity in the webpage (for example, within the appropriate context Chelsea F.C. is the same as Chelsea or Barack Obama is the same as Obama).
  • Make sure to copy and paste the complete URL into your browser to ensure accuracy.
  • We detect improper responses using several sophisticated validators. This can effect your trust rating.

NOT TO DO

  • Be diligent to increase your reputation and trust rating, do not answer questions randomly.
